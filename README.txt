복약 알림 — 웹앱 (약 시간 알림 · 체크)
========================================

[하는 일]
  1. 약 이름 + 먹을 시간 등록 (하루 여러 번 가능)
  2. 오늘 목록에서 먹었으면 줄 눌러 체크
  3. 정해진 시간에 브라우저 알림 (허용 필요)
  4. 자정 지나면 오늘 체크는 새로 시작

[데이터]
  이 기기 브라우저에만 저장 (서버 없음)

[미리보기]
  시작.bat → http://localhost:8789/

  · 앱을 완전히 닫으면 알림이 안 올 수 있음 (브라우저 한계)
